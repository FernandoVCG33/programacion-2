# Programación 2 - Semestre 1-2025 🚀

Este repositorio almacena los trabajos, prácticas y proyectos desarrollados para la materia de **Programación 2** durante el semestre 1-2025.

## 🎯 Objetivo
El objetivo principal de este repositorio es documentar y mostrar paso a paso el aprendizaje y funcionamiento del lenguaje de programación **Java**, aplicando buenas prácticas de desarrollo de software.

## 📂 Estructura del Repositorio
El repositorio compone una estructura de aprendizaje progresivo, abarcando desde lógica básica (condicionales y ciclos) hasta Programación Orientada a Objetos (POO) y el manejo de interfaces gráficas (GUI).

A continuación, se detallan los módulos principales:

* **`/Practica`**: Ejercicios y fundamentos de programación.
* **`/Futbol`** y **`/Medallas`**: Proyectos introductorios aplicando conceptos de Programación Orientada a Objetos (Clases, Objetos, Atributos).
* **`/Ajedrez`**: Proyecto intermedio que aplica lógica compleja, herencia y polimorfismo mediante el diseño de piezas y movimientos.
* **`/interfaces_graficas`**: Ejercicios enfocados en el diseño de ventanas y manejo de eventos de usuario (incluye un sistema de **Elecciones**).
* **`/Proyecto final`**: Sistema integral para un consultorio odontológico, que combina POO, persistencia de datos (manejo de archivos) e Interfaces Gráficas.
* **`/Examenes`**: Evaluaciones y resoluciones de pruebas de la materia.

## 🛠️ Tecnologías y Herramientas Utilizadas
* **Lenguaje:** Java
* **Paradigma:** Programación Orientada a Objetos (POO)
* **GUI:** Java Swing / AWT
* **IDE Recomendado:** Eclipse / IntelliJ IDEA
